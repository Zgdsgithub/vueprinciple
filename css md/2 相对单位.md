# 2.1 相对单位
em：1em等于当前字号（1*当前字体大小）用来设置边距（用在字符上很复杂）通常情况下字符和边框大小是相对的
rem：即root em，相对于根元素字号的大小，这样就可以得到一个确定值
拿不准的时候，用rem设置字号，用px设置边框，用em设置其他大部分属性。

# 视口单位
vh与vw，利用calc结合不同单位的像素计算获得平滑过渡。（直接使用vh字体会过大过小）

# css变量
用--加任意字符表示，跟js一样有作用域，也可用JS动态更改
