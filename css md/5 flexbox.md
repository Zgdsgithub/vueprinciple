# flex属性
flex属性是简写方式，包含三个属性grow、shrink、basis
basis默认值是总宽度的20%,grow增长因子会把剩余的空间占据，grow的值是分子，grow的和是分母
shrink即收缩

# 常用属性
justify-content主轴间距、align-content副轴间距、align-items副轴对齐方式
